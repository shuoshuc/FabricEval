FabricEval

Traffic engineering evaluation framework built for hyperscale networks.

Get Started

What is FabricEval?

FabricEval is an evaluation framework that enables users to study the bahavior and performance of traffic engineering (TE) systems. It models all major aspects of a TE system, including network topology, traffic demands, TE formulation/algorithm, and switch configuration. FabricEval is also shipped with production-like data center network topologies and traffic demands based on Jupiter fabrics, as well as the same TE algorithm powering the Orion SDN controller.
