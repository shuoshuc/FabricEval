FabricEval

Traffic engineering evaluation framework built for hyperscale networks.

Get Started

What is FabricEval?

FabricEval is a flexible and efficient evaluation framework that enables users to study the bahavior and performance of various traffic engineering (TE) systems. It models all major aspects of a TE system, including network topology, traffic demands, TE formulation/solving algorithms, and switch configuration. All of these components are packaged as individual modules, allowing easy replacement without affecting the rest of the framework. It can hence be used to answer what-if questions by controlling various aspects of a TE system. FabricEval is also shipped with Google-production-like large-scale data center network topologies and traffic demands as well as the same TE algorithm used in Google's data centers.
